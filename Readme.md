Project: Recipes API

# Tech Stack:
- Node.js
- Express.js
- MongoDB

# How to run:
npm install
npm run dev

# Endpoints:
POST /api/recipes
GET /api/recipes
GET /api/recipes/:id
PUT /api/recipes/:id
DELETE /api/recipes/:id
